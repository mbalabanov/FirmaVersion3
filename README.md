# Firma Version 3
Dieses Repo enthält die Java-Lösung zum dritten Teil des Projekts "Firma".

Zum Starten, bitte `Verwaltung.java` ausführen und auf die Console achten.

`MitarbeiterTest.java` enthält sechs JUnit-Tests.
